## Hosting

AWS Amplify Hosting is a fully managed CI/CD and hosting service for fast, secure, and reliable static and server-side rendered apps that scale with your business. Supports modern web frameworks such as React, Angular, Vue, Next.js Nuxt.js, Gatsby, and more.

The Amplify Hosting documentation is managed on AWS Docs. To learn about hosting features such as custom domains, redirects, etc., please visit the hosting documentation.
